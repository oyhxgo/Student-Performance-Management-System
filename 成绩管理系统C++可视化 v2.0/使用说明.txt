赤色领域成绩管理系统（纯HTML 本地版）

一、运行
1. 使用 Microsoft Edge 或 Google Chrome。
2. 直接双击 index.html。
3. 页面右上角点击“选择数据文件夹”。
4. 选择压缩包内的 data 文件夹。
5. 程序会在 data 文件夹生成 grade_manager_data.json。

二、数据保存
- 增删改成绩后会自动保存。
- 也可以点击右上角“保存”。
- 手动备份会保存到 data/backups 文件夹。
- 不依赖服务器、Qt、C++、MySQL，也不需要 EXE。

三、导入旧成绩
1. 打开“数据管理”。
2. 点击“选择旧成绩文件”。
3. 选择 legacy_data 文件夹中的第一次、第二次考试成绩文件。
4. 可以一次选中两个文件。

四、注意
浏览器不允许网页在未授权时随意写入电脑文件，因此第一次使用必须手动选择 data 文件夹。
如果浏览器不支持“选择数据文件夹”，请换最新版 Edge 或 Chrome。
